# IzotopeRX_MarkerList_to_Reaper_CSV
This script is convert Izotope RX Marker List file to Reaper CSV

# Requirements

Python 3.xx
argparse

```
pip install argparse
```

# How to Use

Type Command.

```
python rxtocsv.py -i your_rx_file.txt -o your_converted_file.csv
```

Then Import file.
